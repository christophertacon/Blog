Static webpages (blogs) with Pelican

Prepare environment

We will use the lubuntu image that you have already seen as a starting point. But it is relatively easy to install Pelican on your machine as well.

Prepare virtual machine

Hint: Refer to the VirtualBox post for guidance on how to setup VirtualBox.

  • Download the lubuntu image and add to VirtualBox

  • Install make gcc

    sudo apt-get install make gcc

  • Install Guest additions

    sudo ./VBoxLinuxAdditions.run

Installing Pelican

  • Install Python package manager pip

    wget https://bootstrap.pypa.io/get-pip.py sudo python get-pip.py

  • Install pelican and markdown

    sudo pip install pelican markdown

Obtaining the webpage repository

  • Install mercurial

    sudo apt-get install mercurial meld

  • Clone the repository

    hg clone http://bitbucket.org/computationalmodelling/computationalmodelling.bitbucket.org blogs
